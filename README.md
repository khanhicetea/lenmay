## LenMay

> Lên là lên là lên mây luôn ...

### Usages

- Step 1 : Install python3 pip

```bash
$ curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
$ python3 get-pip.py
```

- Step 2 : Install **lenmay**

```bash
$ pip install lenmay
```

- Step 3 : Run init

```bash
$ lenmay init
```

- Step 4 : Kickstart your virtualhost

```bash
$ lenmay web
```
